Bubbles
This is also a blank canvas with very abstract background and a bubbly frame, where

Strobe Light
In this sketch when the mouse is moved the color changes based on its position.

ISS
This scetch is based on the open ISS Position API, where the applitude and the
distance between two points of the wave keep changes based on the ISS position.
The blocks in the back changes its size and speed based on the position of the mouse.

#'s Fact
It is also based on an open API that gives facts about random numbers. If the mouse is
clicked on the middle of the sketch it will show a generative triangles
that will add an accent to the sketch.

Crystal Cave
This sketch is inspired by the crystals in the cave. If the mouse is clicked on the cave
